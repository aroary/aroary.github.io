About

I am currently learning to be a fullstack developer. I started with learning JavaScript (specifically with ProcessingJS) from an online course, and picked up HTML and CSS while doing so. Since web development was not really my thing (the designing part), I got into NodeJS and learnt to use Electron, Express, Socket.io and all the other good NPM packages. I learned to use GitHub and the git CLI along with Visual Studio and Visual Studio Code. While learning NodeJS I also picked up Python (with the help of an online course) because it is so popular and easy to learn. After a few years of small NodeJS and Python projects, I decided I wanted to learn another language. After much though and a bit of consultation, decided to take on C++ (with MSBuild), and now that is what I am doing now. In the future, I hope to be able to work with SQL databases, make a website actually look good (not specifically this one) and do whatever all the other good fullstack developers can do.
